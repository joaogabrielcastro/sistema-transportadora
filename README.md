🚛 Sistema de Auxílio para Transportadora

Este sistema foi desenvolvido para auxiliar transportadoras no gerenciamento de caminhões e seus respectivos pneus, oferecendo controle detalhado dos equipamentos, gastos e manutenções.

⚙️ Funcionalidades
🧾 Cadastro de Caminhões e Pneus

Permite o registro completo de caminhões, associando todos os seus pneus.
Com isso, o sistema oferece um controle mais eficiente e preciso dos equipamentos utilizados, facilitando a gestão da frota.

💰 Controle de Gastos e Manutenção

A área de manutenção possibilita que o responsável registre todos os custos relacionados a um caminhão específico, incluindo peças, serviços e trocas de pneus.
Dessa forma, é possível ter uma visão clara e detalhada dos gastos reais com cada veículo da frota.

🛠️ Tecnologias Utilizadas

Frontend: JavaScript + Vite

Banco de Dados: PostgreSQL no Supabase

Deploy:

Frontend hospedado na Vercel

Backend hospedado na Render

🚀 Objetivo do Projeto

Fornecer uma ferramenta prática e intuitiva que ajude transportadoras a otimizar o controle de manutenção e reduzir custos operacionais, oferecendo dados precisos para tomadas de decisão.
